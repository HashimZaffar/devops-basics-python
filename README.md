# DevOps Basics with Python Automation

This project demonstrates basic DevOps concepts using:
- Linux/macOS terminal
- Python scripting
- Automation mindset

## What This Project Does
- Creates a directory using Python
- Creates a file inside the directory
- Writes content automatically

## Technologies Used
- Python 3
- macOS Terminal (Unix-based)
- os module

## How to Run
1. Clone the repository
2. Run the script:
   python3 file_manager.py

## Why This Project Matters
Automation is a core DevOps skill. This project shows how repetitive tasks
can be automated using simple Python scripts.
